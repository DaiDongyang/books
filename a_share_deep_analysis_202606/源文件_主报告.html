    • 1. 格雷厄姆价值投资框架

    1.1 本杰明·格雷厄姆的核心标准

    在《聪明的投资者》（The Intelligent Investor, 1949/1973）中，格雷厄姆提出了防御型投资者的选股标准：

    标准格雷厄姆原始要求本报告采用的标准说明
    ---------------------------------------------
    市盈率< 15倍< 15倍基于过去3年平均利润或TTM
    市净率< 1.5倍< 1.5倍或 PE × PB < 22.5
    股息率持续分红20年以上股息率 > 3%反映盈利真实性
    财务稳健流动比率 > 2:1资产负债率 < 60%（非金融）；对金融股使用调整标准工业企业的标准
    盈利稳定性过去10年无亏损年份过去10年连续盈利排除周期股盈利高峰假象
    盈利增长10年内每股盈利增长 > 33%10年利润复合增长 > 0%确保不是萎缩中的公司
    公司规模足够大市值 > 500亿人民币排除微型股

    1.2 对金融股的特殊调整

    格雷厄姆的原始标准专为工业企业设计。银行靠杠杆经营（资产负债率90%+），市净率普遍低于1倍，股息率高。如果严格套用原始标准，95%的A股都会被淘汰——不是因为它们都不好，而是中国经济结构决定了金融行业集中了最多符合"低估"特征的公司。

    针对金融股（银行、保险、券商），本报告采用调整后的评分标准：

    标准金融股调整标准满分
    -------------------------
    PE< 10倍 = 1分1
    PB< 1.0倍 = 1分1
    股息率> 4% = 1分1
    ROE> 2%（银行）/ > 10%（保险券商）= 1分1
    10年连续盈利Yes = 1分1
    总分5分

    1.3 格雷厄姆评分解读

    得分含义投资含义
    ---------------------
    5/5完全符合格雷厄姆调整后标准深度价值，安全边际充足
    4/5大部分符合，一项稍弱价值型，值得重点研究
    3/5部分符合具有一定价值特征，但有明显短板
    2/5勉强有2项符合不属于传统价值股
    1/5仅1项勉强符合成长股或投机股
    0/5完全不符合高估值成长股或亏损股，格雷厄姆不会碰

    2. 银行板块

    核心特征: 银行是中国A股的"价值之锚"。全板块PE 3-8倍，PB 0.2-0.9倍，股息率3-7%。几乎所有银行都符合PE<15和PB<1.5的格雷厄姆筛选标准。但银行的高杠杆是其商业模式的核心，不能用工业企业的债务标准来评判。

    行业级因果传导总览

    利率政策传导链：
央行降息（LPR↓10bp）
  → 贷款重定价（存量贷款约70%在1年内重定价）
  → 银行生息资产收益率↓约5-8bp
  → 存款利率同步下调但幅度更小（存款竞争）
  → 净息差（NIM）压缩约3-5bp
  → 对于一家净息差1.5%的银行，利润直接减少约2-3%
  → 股价反应：-3%到-8%（1.5-2倍利润弹性）

    
地产周期传导链：
房价↓10%
  → 开发商抵押物价值缩水
  → 银行房地产开发贷不良率↑约50-100bp
  → 增加拨备计提（贷款损失准备↑）
  → 利润↓约5-15%（取决于地产敞口）
  → 但个人按揭贷款不良率极低（<0.5%），因为中国购房者首付比例高+个人无限责任

    
汇率传导链（对银行）：
RMB贬值3%
  → 外币资产（主要是美元贷款）人民币计价价值↑
  → 但同时外币负债的人民币成本也↑
  → 大多数中资银行外汇敞口很小（<总资产5%）
  → 直接盈利影响：极小（<0.5%）
  → 间接影响：贬值→资本外流→A股下跌→银行股跟跌（β效应）

    2.1 工商银行 (601398) — "宇宙行"

    指标数值格雷厄姆评分
    ------------------------
    PE (TTM)7.6x✅ PE<10 → 1分
    PB0.53x✅ PB<1.0 → 1分
    股息率6.2%✅ >4% → 1分
    ROE9.45%✅ >2% → 1分
    10年连续盈利Yes (20+年)✅ → 1分
    总分5/5 🏆

    资产规模: 约50万亿人民币，全球最大银行 市值: ~2.77万亿人民币

    因果传导链条分析

    链条1：利率↓ → 净息差 → 利润 

    央行降息10bp
  → 工行贷款组合（~26万亿）中约65%-70%为浮动利率或在1年内重定价
  → 贷款收益率↓约7-9bp（1年内影响约18万亿贷款）
  → 同时存款成本↓约3-5bp（存款利率自律机制下，国有大行存款利率下调最慢——因为工行网点最多，零售存款占比高，竞争压力小）
  → 净息差净压缩约4-6bp
  → 对净利润的直接影响：约120-180亿（基于26万亿贷款）
  → 占2025年净利润3,686亿的约3-5%
  → 工行相对优势：存款成本最低（~1.5%），NIM压缩后仍高于行业均值

    链条2：地产风险暴露 → 资产质量 → 拨备 

    房地产行业违约率↑
  → 工行对公房地产贷款约8000亿（占总贷款约3%，行业最低水平之一）
  → 不良率可能从2.5%升至4-5%
  → 新增不良约120-200亿
  → 拨备覆盖率（当前约210%）→ 需增加拨备约60-100亿
  → 对利润影响：约2-3%
  → 结论：工行地产敞口在国有大行中最低，风险可控

    链条3：地方政府债务 → 隐性风险 

    地方政府土地出让收入↓（房地产下行）
  → 地方融资平台（LGFV）偿债能力下降
  → 工行持有LGFV贷款约3-4万亿（含债券）
  → 大规模展期/降息（中央指导下的"化债"）
  → 贷款收益率从4-5%降至2.5-3%
  → 年化利息收入减少约300-500亿
  → 但这个影响在3-5年内分摊（不是一次性冲击）
  → 工行应对：通过扩大中小企业贷款和消费贷来对冲

    链条4：汇率波动 → 北向资金 → 股价 

    RMB兑美元贬值2%
  → 境外投资者（持有工行A股约3-5%）的以美元计价回报减少2%
  → MSCI中国指数权重被动调整
  → 北向资金净卖出工行（或减少买入）
  → 工行股价短期承压1-3%
  → 但因为工行H股同时存在（AH溢价约20-30%），部分北向资金会从A股切换到更便宜的H股
  → 长期来看：工行的外资持股比例（~3%）远低于招行（~8-10%），汇率敏感度中等偏低

    跨资产关联:

  • 房地产: 正相关但弱于预期。地产↓→LGFV风险↑→工行承压。但地产政策放松（降首付、降利率）→工行按揭贷款量↑
  • 黄金: 弱负相关。黄金↑=避险情绪↑=资金流出银行股。但工行稳定高股息在避险环境中部分对冲
  • 比特币: 极低相关。比特币波动对工行几乎无影响

    指标数值格雷厄姆评分
    ------------------------
    PE5.9x✅ 1分
    PB0.53x✅ 1分
    股息率5.6%✅ 1分
    ROE10.04%✅ 1分
    10年连续盈利Yes✅ 1分
    总分5/5 🏆

    特色: 建行是中国最大的住房按揭贷款银行（房贷占总贷款约30%），也是基础设施贷款（基建贷款占对公贷款约40%）的最大提供者。

    因果传导链条

    链条1：房地产政策 → 按揭贷款 → 建行（最核心传导链） 

    政府放松房地产政策（降首付至15%、降按揭利率至3.0%以下）
  → 购房者月供减少→购房意愿↑
  → 新建商品房销售面积↑（政策传导到销售需3-6个月）
  → 建行按揭贷款发放量↑（建行在按揭市场份额约15%，行业最大）
  → 但存量按揭贷款重定价（LPR下调）
  → 新增按揭3.0%利率 vs 存量按揭平均4.0%
  → 新增贷款对净息差仍是稀释而非增厚
  → 建行净息差从1.6%缓慢下降至1.5%左右（年化下降约5bp）
  → 结论：以量补价的逻辑在按揭贷款上只能部分成立——新增2000亿3%贷款无法弥补2万亿存量贷款降息50bp的损失

    链条2：基础设施投资 → 对公贷款 → 利润 

    政府加大专项债发行（2025年约4万亿）
  → 基建项目资本金到位
  → 建行配套项目贷款（公路、铁路、水利）
  → 对公中长期贷款增长约8-10%
  → 基建贷款收益率约3.2-3.5%（低于普惠贷款但风险低）
  → 利润贡献稳定但利润率薄
  → 建行在基建贷款市场排名第二（仅次于国开行政策属性贷款）

    链条3：居民存款行为 → 负债成本 → 净息差 

    居民消费/购房意愿低迷
  → 居民储蓄率↑（2024-2025年储蓄率创近年新高）
  → 定期存款占比↑（活期存款转向定期）
  → 定期存款利率约1.5-2.0% vs 活期存款利率0.2%
  → 存款成本↑约10-15bp
  → 这是当前银行净息差压缩的主要驱动力——贷款降息的同时存款反而变贵了
  → 建行零售存款占比高（~50%零售），对存款定期化趋势暴露较大

    2.3 招商银行 (600036)

    指标数值格雷厄姆评分
    ------------------------
    PE6.4x✅ 1分
    PB0.78x✅ 1分
    股息率5.2%✅ 1分
    ROE13.44%✅ 1分

    总分5/5 🏆

    特色: 中国零售银行之王。零售AUM（管理客户资产）超过14万亿，超过其表内资产规模。私人银行客户数中国第一。

    因果传导链条

    链条1：资本市场活跃度 → 财富管理中收 → 利润结构 

    A股成交量↑100%（牛市情景）
  → 招行代销基金规模↑（招行是最大的基金代销银行，市占率约10-12%）
  → 基金代销手续费↑
  → 同时：代销保险、信托、理财等财富管理收入↑
  → 财富管理中收（净手续费收入）可能增长20-30%
  → 财富管理收入占招行营收约30%（vs 行业平均10%）
  → 利润弹性显著高于同业（"银行中的券商"）
  → 但反过来，熊市→基金赎回潮→中收崩塌→利润双杀

    链条2：利率↓ → 负债优势 → 竞争优势 

    央行降息周期
  → 行业净息差普遍压缩
  → 招行活期存款占比约60%（vs 行业平均40%）
  → 活期存款利率仅0.2%，几乎不受降息影响
  → 定期存款利率随政策下调→招行的定期占比低→受影响更小
  → 存款成本仅约1.2-1.3%（行业最低之一）
  → 净息差虽也下降，但降幅小于行业平均→竞争优势反而扩大
  → 这是招行最深的护城河：低成本核心存款

    链条3：汇率 → 外资持股 → 股价波动 

    RMB贬值→境外投资者对中国金融股的美元计价回报下降
  → 招行是外资持股比例最高的中资银行（北向资金持股约8-10%）
  → 外资抛售招行的幅度大于其他银行
  → 股价跌幅大于工商银行等国有大行
  → 历史上：2023年8月-2024年1月人民币贬值期间，招行跌幅约20%，工行跌幅约8%
  → 汇率是对招行股价最敏感的外部变量之一

    链条4：消费信贷 → 信用卡/消费贷 → 利润 

    经济下行→居民收入预期转弱→消费意愿↓
  → 信用卡交易量增速放缓
  → 信用卡分期/循环信贷余额↓
  → 信用卡不良率可能从1.5%升至2-3%
  → 招行信用卡业务（行业前三）利润贡献约10-15%
  → 消费降级对招行的冲击大于国有大行（招行零售占比高）
  → 但招行的零售客户质量更高（筛选中高收入客户），抗风险能力强于平安银行等

  → Shibor/DR007维持在1.5-2.0%低位

  → 转型过程中ROE从15%降到10%→继续降到9%
  → 但PB 0.44倍已经反映了市场对ROE下降的悲观预期

    链条2：绿色金融政策 → 绿色信贷 → 增长 

    双碳政策推进→清洁能源/节能减排项目需求↑
  → 兴业绿色信贷余额超过2万亿（行业第一）
  → 央行对绿色信贷给予定向降准/再贷款支持→获取低成本央行资金
  → 绿色信贷收益率虽低（约3.0-3.5%），但资本占用低
  → 兴业绿色信贷增速约20-25%/年
  → 这是兴业在净息差压缩中维持利润增长的核心引擎
  → 政策风险：如果经济下行，双碳推进可能减速

    2.10 银行板块格雷厄姆筛选总结

    全部20家银行均符合PE<15和PB<1.5的原始格雷厄姆标准。

    按调整后5分制，12家银行获得5/5满分，6家获得4/5，仅2家（民生、华夏）低于4分。

    关键矛盾：银行的杠杆率（资产负债率90%+）不符合格雷厄姆"低负债"要求。对此的经典回应：

  • 银行的"负债"是存款，不是债务——存款是银行的原材料而非负担
  • 银行受央行/银保监会的严格监管（资本充足率、流动性覆盖率等），这些监管指标是比"资产负债率"更好的安全度量
  • 格雷厄姆本人也承认，其工业企业标准不完全适用于金融公司

    • 格雷厄姆会买的银行（如果他是中国投资者）：招商银行（最优零售银行+最低存款成本）、建设银行（最强房贷+基建）、交通银行（PB 0.48，以不到一半的净资产价格买入）、中信银行（7%股息率）

    9. 大类资产关联矩阵

    9.1 各类资产与中国A股的相关性机制

    股票 vs 房地产

    传导机制（6条路径）：

    
路径1：信贷渠道
  央行宽松→信贷扩张→同时流入楼市和股市→两者同涨
  2015-2016年最为明显：降息降准→深圳房价翻倍+创业板牛市

    
路径2：财富效应渠道
  房价↑→家庭资产负债表改善→消费信心↑→可选消费（白酒、家电、汽车）盈利↑→相关股票↑
  反之：2021年至今房价下跌→财富负效应→消费降级→茅台/五粮液/美的盈利承压

    
路径3：替代效应渠道
  房价预期回报率↓（从10%+/年降到0%或负）→投资性购房需求↓
  →居民储蓄寻找替代资产→部分流入股市（但大部分先去了理财/存款）
  →这个渠道目前还在"初期"——大规模存款搬家需要股市赚钱效应配合

    
路径4：银行资产质量渠道
  房价↓→开发商贷款不良↑→银行拨备↑→银行利润↓→银行股↓
  这是银行股与房地产最直接的相关性

    
路径5：地方政府财政渠道
  土地出让金↓50%+→地方财政收入锐减→削减产业补贴/基建投资
  →科技补贴减少→半导体/新能源股票承压
  →基建投资减少→水泥/钢铁/建筑股票承压

    
路径6：建材需求渠道
  房地产新开工↓50%+→水泥/钢铁/玻璃/铝材需求↓→相关股票盈利↓

    综合判断：A股与房地产的正相关性在减弱，但远未消失。银行（资产质量）、建材（需求）、消费（财富效应）三个渠道仍然高度绑定房地产。

    黄金定价的三个核心变量：

  • 美国实际利率（TIPS收益率）= 名义利率 - 通胀预期
    • 
   实际利率↓（降息或通胀↑）→黄金↑
   

  • 美元指数（DXY）
    • 
   美元↓→以美元计价的黄金↑
   

  • 全球风险偏好/地缘政治
    • 
   避险情绪↑→黄金↑

    
A股与黄金的相关性逻辑：

  • 在经济"好"的时候（复苏期）：A股↑ + 黄金↓（资金从避险资产流向风险资产）
    • 
  → 负相关

  • 在经济"坏"的时候（滞胀/危机）：A股↓ + 黄金↑ 
    • 
  → 负相关

  • 在流动性泛滥的时候（QE）：A股↑ + 黄金↑（都受益于流动性）
    • 
  → 正相关

  • 结论：A股与黄金的相关性不稳定，取决于当前的主导因素。

  • 山东黄金（纯金矿，黄金每涨10%，净利润约增25-30%）

    10. 结论与配置建议

    10.1 核心结论

  • 格雷厄姆价值投资的天然猎场是中国的银行股。 全部20家上市银行PE均<10倍，PB最低仅0.24倍（民生），股息率最高7.1%（中信银行）。如果格雷厄姆今天来中国，他很可能把50%以上的仓位配在银行。

  • 但银行的高杠杆是格雷厄姆框架的致命矛盾。 资产负债率90%+在工业企业是无法接受的。回答这个矛盾需要理解：银行的"负债"是存款（原材料），不是债务（负担）。监管资本充足率>10.5%的要求比任何"低负债率"都更能保护投资者。

  • 格力电器是当前沪深300中唯一完全符合格雷厄姆原教旨标准的非金融公司——PE 8.2x（<15），PB 1.50（≤1.5），股息率7.4%（>3%），10年+连续盈利，资产负债率约60%（<60%的边界）。洋河股份（PE 6.59，PB 1.31）是唯二。

  • 科技/AI板块零符合格雷厄姆标准。 但这不意味着它们不值得投资——这意味着它们属于完全不同的投资哲学（费雪/成长股投资）。当前沪深300的权重集中在科技股（中际旭创4.83%），如果AI资本开支周期见顶，指数可能面临重大回调。

  • 大类资产之间，A股价值股（银行/家电/公用事业）与黄金形成天然对冲——前者受益于经济复苏+低通胀，后者受益于滞胀+高通胀。组合搭配可以显著降低波动。

    10.3 关键风险

  • 银行净息差尚未见底。如果LPR进一步下调而存款利率不跟，2026-2027年银行利润可能继续承压。
  • AI资本开支周期风险。中际旭创、新易盛当前PE 60-70倍建立在AI算力需求持续高增长的假设上。一旦增速见顶，估值可能腰斩。
  • 房地产风险尚未完全出清。部分银行的房地产不良可能在2026-2027年继续暴露。
  • 地缘政治。中美关系、台海局势是最大的尾部风险——一旦升级，所有中国资产（无论价值还是成长、股票还是房产）都会同步下跌。
