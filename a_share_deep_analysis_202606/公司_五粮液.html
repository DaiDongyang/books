房价上涨（2020年前） → 居民净资产增加 → "财富效应"使家庭消费意愿上升 → 高端白酒（茅台/五粮液）、高端家电（卡萨帝）、医美（爱美客）、旅游免税（中免）等消费全面受益。 房价下跌（2021年后） → 居民净资产缩水 → 财富效应逆转 → 消费降级涌现 → 可选消费全面承压 → 只有必需消费品（海天酱油、999感冒灵）受影响较小。

    5. 泸州老窖 (000568) -- 高端挺价 vs 现实需求

    泸州老窖的核心战略矛盾在于：高度国窖1573（52度）作为对标五粮液的高端产品，在需求走弱环境下，公司选择"控货挺价"——宁愿销量下降也不降价。这一战略的逻辑是：消费降级环境下，白酒品牌的"价格锚"一旦下降，品牌定位的修复需要数年甚至无法修复。因此泸州老窖在2025年大幅削减高度国窖的发货量，导致中高档酒营收229.7亿元（-16.7%）。因果链为：经济下行 → 高端白酒需求收缩 → 非刚性需求客户转向低价替代品 → 泸州老窖选择控量保价 → Q4营收同比-62.2% → 维护了国窖1573的价格体系但牺牲了短期业绩。作为对冲，低度国窖（38度）在华东等区域快速扩张，部分弥补了高度产品的下滑。

    2. 五粮液 (000858) -- 主动出清的深度调整者

    五粮液2025年的关键变化在于会计核算方式的调整。公司基于谨慎性原则，在年报中新增"监管商品"（49.1亿元）和"监管商品款项"（263.1亿元）科目，仅确认已实现终端动销的出货收入。这意味着大量已发给经销商但尚未被终端消费者购买的商品不再立即确认为收入。这一调整的直接后果是营收从正常水平（约800亿元量级）骤降至405亿元。但这并非终端需求的断崖式下跌，而是会计确认节奏的变化。宏观经济传导机制在此处被会计准则"扭曲"：经济下行 → 终端动销放缓 → 经销商库存去化周期延长 → 公司无法确认"压货"收入 → 表观数据恶化 → 但若终端需求边际改善，2026年将迎来确认收入的"补偿性"增长。2026年Q1营收228亿元（+33.67%）、净利润80.6亿元（+82.57%）的强势反弹印证了这一判断。在2024-2025年的行业下行周期中，五粮液核心产品普五（八代）面临批价倒挂压力——出厂价约969元/瓶，批发价一度跌至不足900元/瓶。这导致经销商每卖一瓶五粮液就亏损数十元，渠道推力急剧下降。因果链为：经济放缓 → 需求走弱 → 批价承压 → 经销商利润转负 → 经销商减少回款或不推高 → 动销进一步放缓。五粮液在2025年主动降低打款价、加大渠道返利、不强制回款，以"刮骨疗伤"的态度修复渠道生态。2026年春节动销数据（五粮液产品动销1.4万吨，同比增长17%，终端进货商家数量同比+60%）表明渠道信心修复的效果正在显现。
