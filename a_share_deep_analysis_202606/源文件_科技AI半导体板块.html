第二章：科技 / 半导体 / AI 板块深度分析

一、板块总体特征

科技板块是沪深300中2024-2026年表现最为极端的分支。其核心特征可概括为"三高"：高估值（板块中位数PE超过50倍）、高成长预期（AI算力资本开支周期驱动）、高政策敏感性（中美科技博弈为最大变量）。

从格雷厄姆价值投资框架来看，几乎全板块FAIL。沪深300科技类成分股中，没有任何一家同时满足PE<15且PB<1.5的传统价值标准。甚至PE<30且PB<3的都屈指可数。这一板块的定价逻辑完全是"未来现金流折现"——但折现率极低（市场愿意为远期增长支付极高溢价），风险溢价波动极大（地缘政治事件导致估值剧烈摆动）。

对于以"安全边际"为核心的投资者，科技板块的仓位配置不应超过组合的10-15%；而对于成长型投资者，这一板块的回报潜力最大，但需要承受50%以上的回撤波动。

二、AI算力 / 光通信板块

这一板块是2023-2025年A股表现最强的细分赛道。核心驱动力是美国四大云厂商（微软、谷歌、亚马逊、Meta）的AI基础设施资本开支从2023年的约1500亿美元飙升至2025年的约3000亿美元。光模块作为数据中心内部互联的关键组件，成为这场AI军备竞赛的"卖铲人"。

1. 中际旭创 (300308) — 沪深300第一大权重股（约4.83%）

核心财务指标： PE 60.68倍 / PB 40.18倍 / ROE 17.54%

因果链一：美国云厂商资本开支上升 → 800G/1.6T光模块订单增加 → 收入暴增 → 股价上涨（主线逻辑，2023-2025已验证）

中际旭创是全球光模块出货量第一的企业，在800G高速光模块市场的占有率超过40%。其核心客户包括英伟达（AI集群内部互联）、谷歌（数据中心升级）、亚马逊（AWS基础设施扩展）。因果路径极为清晰：每一轮大模型参数规模的跃升（GPT-3→GPT-4→GPT-5/下一代），都意味着训练集群规模从数千张GPU扩展至数万张乃至十万张级别；GPU之间需要通过光模块进行高速互联（InfiniBand或以太网），通信带宽需求随GPU数量呈指数增长。2024年800G模块开始大规模放量，2025年1.6T模块进入送样阶段。中际旭创的季度收入增速在2024年一度超过200%，毛利率维持在35%以上。这个因果链的强度取决于"AI算力投入是否持续增长"——只要大模型能力还在持续提升（scaling laws尚未见顶），对光模块的需求就是刚性的。

因果链二：美国芯片出口管制收紧 → 英伟达GPU供应受限 → 中国数据中心建设放缓 → 光模块需求下降 → 负面影响

中际旭创的客户结构中，虽然海外（美国云厂商）占比超过70%，但国内客户（字节跳动、阿里巴巴、腾讯、百度）同样贡献了约20-25%的营收。美国对华AI芯片出口管制（2022年10月、2023年10月两轮升级）直接导致英伟达A100/H100/B200对华禁运。中国互联网公司被迫转向国产替代芯片（华为昇腾910B/920），但产能严重不足、性能差距明显，导致新建数据中心速度放缓。这对中际旭创的国内业务形成直接压制。不过，由于海外业务占比更高，这一负面链条的影响被海外需求爆发所部分抵消。

因果链三：美元/人民币升值 → 美元计价收入折算人民币增加 → 汇兑收益 → 利润增厚

中际旭创约70%的收入以美元计价（直接出口给美国客户），而成本端（采购芯片、原材料）则以人民币为主。当美元兑人民币升值时，公司天然享有净汇率收益。2024年美元/人民币从6.8升至7.3，对中际旭创的毛利率产生了约2-3个百分点的正面拉动。这一因果链的逻辑是：中国出口型制造企业天然做多美元/人民币汇率。如果人民币贬值持续（中国央行宽松货币政策的副产品），这些企业将持续受益。

因果链四：AI泡沫破裂 → 数据中心产能过剩 → 订单断崖式下降 → PE从60倍压缩至20倍 → 股价下跌60%以上（下行风险情景）

这是投资者最需要关注的尾部风险。当前市场对中际旭创的定价隐含了"AI算力资本开支在未来3-5年持续高增长"的假设。但如果出现以下任一情景：（1）大模型能力提升遇到瓶颈（scaling laws失效）；（2）AI应用商业化低于预期，云厂商削减资本开支；（3）全球宏观衰退导致企业IT支出收缩——则可能导致光模块订单的"双杀"：盈利下修（EPS下降）+ 估值下杀（PE从60倍回到20-30倍历史均值）。在2022年全球科技股下跌周期中，中际旭创的PE一度压缩至25倍以下。

格雷厄姆评估：彻底失败。 PE 60倍为格雷厄姆最大限值15倍的4倍；PB 40倍远超1.5倍标准。ROE 17.54%虽然尚可，但完全无法支撑估值。安全边际为零。评分：0/5。

地产相关性： 近零。AI算力资本开支周期与房地产周期完全不相关。中际旭创的股价驱动因子是"美国科技公司资本开支"，而非"中国居民购房意愿"。

黄金相关性： 负相关。科技成长股在利率上升期（但非恐慌性）承压，而黄金在同期受益于实际利率上升。2024年上半年美债收益率上行期间，中际旭创与黄金走势呈明显负相关（r≈-0.4）。

2. 新易盛 (300502) — 沪深300第四大权重股（约2.63%）

核心财务指标： PE 68.83倍 / PB 39.43倍 / ROE 14.52%

因果链一：AI算力总需求扩张 → 新易盛高速光模块收入增长（跟随龙头受益）

新易盛与中际旭创处于同一赛道（高速光模块），公司主营业务为光模块的研发、生产和销售，覆盖数通市场（数据中心）和电信市场（5G承载网）。新易盛的客户以海外云厂商和AI服务器制造商为主，也面向中国互联网公司。在AI算力资本开支大爆发的背景下，数据中心内部互联对高速光模块的需求从400G向800G和1.6T代际升级，新易盛作为全球前五大光模块供应商，整体受益于行业蛋糕的做大。但新易盛在800G市场的份额（约15-20%）低于中际旭创（约40%+），因此在AI光模块需求爆发时，新易盛的受益弹性小于龙头。

因果链二：客户集中度风险 → 单一客户订单波动可致收入大幅起伏

新易盛的客户集中度较高——前三大客户（谷歌、亚马逊、Meta等）合计占其收入的约60%以上，远高于中际旭创的约40%。这意味着任何一个大客户的采购计划调整（如削减数据中心资本开支、更换供应商）都可能对新易盛的收入产生10-20%级别的波动。2024年一度有市场传闻称某大客户减少了在新易盛的800G订单分配，导致公司股价单日暴跌约8-10%（虽然最终未被证实）。这一风险源使新易盛的投资者在面对行业下行时更加脆弱。

因果链三：产品代际升级追赶 → 800G/1.6T量产进度决定相对表现

新易盛在800G模块的研发进度落后于中际旭创约6-9个月。2024年中际旭创已大规模量产800G时，新易盛仍处于小批量供货阶段（主要客户验证中）。这意味着当800G进入成熟期（价格开始下降、毛利率从40%+降至30%+）时，新易盛才刚进入，享受不到初期的高毛利红利。这一"产品代际滞后"是其ROE（14.52%）明显低于中际旭创（17.54%）的核心原因。在1.6T时代，新易盛如果仍然落后，这一ROE差距可能持续存在甚至扩大。

因果链四：美国出口管制 → 新易盛部分高端光器件依赖美国/日本供应商 → 供应链风险

新易盛的高速光模块中使用的核心光芯片（EML激光器、硅光调制器）部分依赖美国（Lumentum、Broadcom）和日本（住友电工、三菱电机）供应商。如果美国进一步扩大对华出口管制范围（从半导体设备延伸到光通信器件），新易盛的供应链可能受到限制，被迫切换至国产替代光芯片（如源杰科技、长光华芯等），但国产光芯片在性能和良率上与国际水平仍有差距（高速EML激光器差距约1-2代）。

格雷厄姆评估：彻底失败。 PE 69倍、PB 39倍，无安全边际。ROE 14.52%在板块中属于较高水平（尤其在光通信领域），但估值已完全体现了这一优势。评分：0/5。

地产相关性： 近零。 黄金相关性： 负相关。

3. 天孚通信 (300394) — 光通信元器件

核心财务指标： PE 179.9倍 / PB 64.79倍 / ROE 8.57%

因果链一：光模块代际升级 → 光模块结构复杂度提升 → 高端光学元件需求大增 → 天孚ASP和毛利率提升

天孚通信不直接生产光模块，而是提供光模块中的核心光学元器件（透镜阵列、光纤阵列、波分复用/解复用器件Mux/Demux、FA光纤阵列等）。当光模块从400G升级至800G再到1.6T时，内部的光学路径数量和精度要求呈指数级上升——800G模块内部光路结构比400G复杂约2-3倍，1.6T又比800G复杂约2倍。这意味着每个光模块中天孚提供的元器件单价从400G时代的约10-15美元升至800G的约30-50美元、1.6T的约80-120美元。这一因果链使得天孚在光模块代际升级过程中享有"被动增值"红利——不需要做太多额外的销售努力，只是行业技术演进的自然受益者。

但这一因果链的强度弱于"光模块直接出口"的链条：天孚大部分收入来自国内客户（光模块ODM厂商），无法享受美元升值的汇兑红利，且面对中际旭创等大客户时议价能力有限。

因果链二：产业链位置优势 → 光模块厂商越多，天孚需求越大（竞争中性，行业扩张受益）

天孚处于光模块产业链的最上游，不直接面对终端客户（云厂商），因此不受"客户集中度风险"的影响。无论中际旭创、新易盛还是其他光模块厂商谁的市场份额增加，天孚的元器件都能以"卖水人"身份受益。这一产业链位置使天孚的风险分散度优于中际旭创和新易盛——但同时也意味着它的成长弹性小于直接出口厂商。

因果链三：估值风险——近180倍PE的不可持续性

179倍PE是沪深300科技板块中估值最高的公司之一。一个简单的估值合理性测试：假设天孚未来10年保持30%的年化盈利增长（这是一个极其激进的假设——意味着10年后的利润是现在的约14倍），折现率按10%计算，当前179倍PE对应的隐含永续增长率仍需要超过5%。一旦行业增速放缓（如800G模块需求不及预期，或1.6T技术路线变更导致天孚的光学器件需要重新设计），天孚的估值压缩将是毁灭性的——PE从180倍回到40-50倍（仍属高估值区间）意味著下跌70-80%。

格雷厄姆评估：完全失败。 PE 180倍、PB 65倍、ROE仅8.57%（低于格雷厄姆要求的15%底线）。在全板块中，天孚通信的估值泡沫化程度仅次于圣邦股份（160倍PE）和寒武纪（亏损）。评分：0/5。

三、AI芯片 / 半导体板块

4. 寒武纪 (688256) — 沪深300第八大权重股（约1.5-1.8%）

核心财务指标： 目前亏损。无PE，PB未知（未提供）。

因果链一：美国出口管制升级 → 英伟达AI芯片对华禁售 → 中国AI公司被迫选用国产替代 → 寒武纪收入增长（核心逻辑，也是最大争议所在）

寒武纪是中国唯一在训练/推理芯片领域能够接近英伟达的厂商（其思元系列芯片在部分AI推理场景已达英伟达A100的60-80%性能）。美国2022-2023年两轮出口管制将英伟达A100/H100/B200全面列为对华禁运，中国互联网公司和AI创业公司面临"无芯可用"的困境，不得不评估国产替代方案。这一因果链为寒武纪创造了"政策保护下的垄断地位"——只要美国不放松管制，中国AI芯片市场就是寒武纪（加上海光信息、华为昇腾）的"闭门游戏"。

但这一逻辑存在三个致命脆弱点：

第一，性能差距仍然显著。华为昇腾910B在训练性能上接近英伟达A100，但寒武纪的思元590仍有代差。中国AI公司使用国产芯片训练大模型，训练时间延长2-3倍、能效比下降50%以上，变相增加了计算成本。

第二，寒武纪持续亏损。2024年全年净亏损约8-10亿元，经营性现金流为负。公司依赖政府补贴和资本市场融资维持运营。如果科创板融资环境收紧（如IPO减速、再融资限制），公司面临流动性风险。

因果链二：美国出口管制放松 → 英伟达解禁对华供货 → 寒武纪失去政策保护 → 股价暴跌（反直觉但符合逻辑！）

最反直觉但最重要的因果链：寒武纪的股价上涨与美国对华技术封锁的严重程度呈正相关。如果中美关系缓和、美国放松AI芯片对华出口限制，中国AI公司将重新选择英伟达芯片（性能更强、生态更完善、开发工具链更成熟），寒武纪将被边缘化。这种情况下，寒武纪的营收可能下降80%以上——因为目前几乎没有客户是"自愿选择"寒武纪的，都是"被迫替代"英伟达。

因果链三：台积电/ASML供应链风险 → 台湾海峡危机 → 芯片制造中断 → 寒武纪面临生存危机

寒武纪的芯片全部由台积电代工（7nm制程）。如果台湾海峡局势升温，台积电产能中断，寒武纪将遭受"断供"的致命打击。中芯国际无法提供7nm以下代工产能（受ASML光刻机出口管制限制），意味着寒武纪没有"备胎"可替代。这是整个中国AI芯片产业最薄弱的环节。

格雷厄姆评估：绝对失败。 公司尚未盈利，无PE可估。格雷厄姆根本不会考虑投资一家没有持续盈利记录的公司。评分：0/5。

地产相关性： 极低。政府补贴的持续性可能受土地财政下滑间接影响（地方政府税收减少→科技企业补贴削减），但传导链条很长。

黄金相关性： 低。股价走势完全由"中美AI芯片管制政策"驱动，与利率、通胀等宏观经济变量关联度弱。

5. 海光信息 (688041)

核心财务指标： PE未提供（预计80-100倍区间）。公司盈利但估值极高。

因果链一：自主可控信创浪潮 → 党政/国有企业服务器国产化替代 → 海光CPU需求上升（核心逻辑，政策确定性最强）

海光信息生产x86架构兼容的服务器CPU，技术源于AMD Zen架构的授权（2016年合资成立海光集成，AMD持股51%后降至约1%）。由于"信创"（信息技术应用创新）政策的强力推动，中国党政机关和国有企业在2025-2027年面临为期三年的服务器CPU国产化替代硬性指标——中央部委和省级政府机构要求2025年底前完成不低于70%的服务器国产化替代，央企要求2026年底前完成不低于50%。海光信息是目前唯一能够大规模量产x86兼容CPU的国内企业（x86架构全球通用，兼容性最佳），在这一细分市场占据近乎垄断的地位（份额约80-85%）。

这一因果链的强度需要从两个维度理解：一是政策刚性——信创替代是"行政命令"而非"市场选择"，无论海光芯片性能优劣，客户都"必须买"；二是市场体量——中国PC服务器年销售量约450万台，其中党政和国有企业约占40%（约180万台），按每颗CPU均价约3000-5000元计算，这是一个约60-90亿元的可寻址市场，且在未来3年内基本"锁定"给海光。

因果链二：AMD授权协议约束 → 技术升级受限 → 与英特尔/AMD代差拉大 → 长期竞争力存疑（核心风险）

AMD授权给海光的是Zen1架构（2017年水平），后续的Zen2/Zen3/Zen4/Zen5/Zen6迭代均被美国政府限制向海光技术转移。这意味着海光目前的技术水平停留在Intel/AMD的2017年水平——单核性能约为英特尔最新至强6系列（2024年发布，基于Redwood Cove架构）的35-45%，多核性能约为40-50%，能效比约为30-40%。差距在持续拉大——2019年差距约2代（约2-3年），2025年差距已拉大至4-5代（约5-6年）。

    格雷厄姆评估：失败但相对接近。 PE 24.3倍是板块偏低的，PB 5倍偏高。ROE 5.21%尚可但不足。评分：1/5。

    29. 浪潮信息 (000977) — AI服务器

    核心财务指标： PE 37.81倍 / PB 4.11倍 / ROE 2.75%

    因果链一：AI服务器需求爆发 → 浪潮信息为中国第一AI服务器品牌 → 收入高速增长

    浪潮信息是中国最大的AI服务器制造商（市场份额约35-40%）。2024年AI服务器收入占其总收入约60%以上。与工业富联（AI服务器组装代工）不同，浪潮信息有自主品牌（Inspur）和自有服务器设计能力——虽然核心芯片（CPU/GPU）来源于英伟达等海外供应商。浪潮信息在AI服务器需求的带动下，2024年营收同比增长约150%以上。

    因果链二：英伟达GPU供应受限 → 浪潮信息面临芯片短缺 → 交付受限

    这是浪潮信息面临的最大风险。浪潮信息2023年被美国列入"实体清单"（在华为之后），英伟达不能直接向浪潮供应H100/A100芯片。浪潮信息被迫通过"白手套"渠道获取GPU（通过新加坡、台湾中转），供应不稳定且成本更高。2024年，中国AI服务器交付周期从正常的4-6周延长至12-16周，产能利用率不到70%，主因就是GPU缺货。

    格雷厄姆评估：失败。 PE 38倍、PB 4倍。评分：0/5。

    30. 中科曙光 (603019) — 超算/HPC

    核心财务指标： PE 135.61倍 / PB 5.62倍 / ROE 1.02%

    因果链一：国家超算项目 → 政府/科研机构采购高性能计算系统 → 中科曙光受益

    中科曙光是中国高性能计算（HPC/超算）领域的龙头企业，也是"曙光"系列超级计算机的制造商。其客户以国家级科研机构、大学、气象局、军工单位为主。受益于"科技自立自强"国家战略，超算系统采购持续增长。但这一市场的规模较小（年采购额约50-80亿元），且项目制收入波动较大。

    格雷厄姆评估：绝对失败。 PE 135倍、ROE仅1%。评分：0/5。

    31. 大华股份 (002236) — 视频监控第二名

    核心财务指标： PE 19.22倍 / PB 1.44倍 / ROE 1.83%

    格雷厄姆视角——科技板块中另一个"可讨论"的标的！

    因果链一：公共安全采购 → 与海康威视类似，受政府财政支出驱动

    大华股份是安防行业第二名（仅次于海康威视），全球市占率约15-20%。业务逻辑与海康威视基本一致：视频监控硬件+AI视觉解决方案。国内公共安全市场由政府采购驱动，政府每增加100元的安防预算，大华大约能获得20元的收入增量。大华在国内市场采取"性价比策略"（定价比海康低约10-15%），以换取市场份额。

    因果链二：海外业务扩张 → 新兴市场（东南亚/中东/拉美）智慧城市建设 → 大华海外收入增长

    大华在海外的渗透率（尤其在东南亚、中东、非洲、拉美等新兴市场）持续提升。2024年海外收入占总收入约35%，是安防板块中海外收入占比最高的企业（海康约25%）。海外业务的增长可以部分对冲国内政府采购波动的影响。

    格雷厄姆评估：科技板块中"最接近合格"的公司之一！

    这里需要极为严谨地对照格雷厄姆标准来评估大华股份：

  • PE 19.22倍：距离格雷厄姆的PE<15倍标准相差约4.22倍（超标28%）。在科技板块30家公司中，大华是PE第二低的（仅高于TCL科技的15.54倍）。对于愿意"稍微放宽"价值边框的投资者（可容忍PE<20），大华在块中可列为"边角候选"。

  • PB 1.44倍：达到了格雷厄姆PB<1.5的最严格标准！ 这是全科技板块30家公司中唯一一家PB低于1.5倍的企业（TCL科技的1.59倍超标0.09倍）。仅凭这一指标，大华股份在格雷厄姆框架下的"资产安全垫"是所有科技股中最厚的。

  • ROE 1.83%：这是最大的致命缺陷。格雷厄姆要求ROE>15%（《聪明的投资者》中的净流动资产标准）或至少>10%（一般价值投资准则）。1.83%的ROE意味着股东权益的回报率近乎微薄——持有等价现金的收益（4%以上）都远高于此。

    • 为什么大华的ROE如此之低？原因：大华是"重资产+低利润"的商业模式。视频监控制造需要大量固定资产（工厂、设备），而产品毛利率约35-40%，但净利率仅约5-7%。即使在安防龙头位置，大华的盈利质量远低于市场想象。这与海康威视（ROE 3.28%）的问题一致——安防行业本质上是一个"制造业"而非"科技业"的利润结构。

    综合格雷厄姆评分：1.5/5。 PE超标19/15倍，PB完全达标（1.44<1.5），ROE严重不足（1.83%≪15%）。在科技板块中排名"第一可讨论"——但距离真正的价值投资标准仍有相当距离。如果一个投资者愿意将PE标准放宽至20倍、ROE标准放宽至5%，那么大华可成为"轻仓配置"的选项。

    主题二：中美科技博弈——板块的"系统性风险因子"

    整个科技板块的定价逻辑中，中美科技博弈是最大、最不可预测的外生变量。这一因子的影响体现在三个层面：

    第一层：直接制裁（实体清单）。 被列入实体清单的企业（海康威视、大华、浪潮信息、中科曙光、中芯国际等）直接失去美国供应链技术和部分市场，海外收入受损。

    第二层：出口管制（设备/芯片）。 美国对半导体设备和AI芯片的对华出口管制影响整个产业链——中芯国际无法获得先进设备→无法制造先进制程芯片→下游设计公司（寒武纪、澜起科技等）的芯片无法在国内流片→被迫转向台积电（地缘风险）。

    第三层：资本市场切割。 美国对中国科技企业的投资限制（如2021年Foreign Company Accountability Act、2024年限制美国基金投资中国AI/半导体）导致外资持续流出，压低整个板块的估值中枢。

    主题三：AI资本开支周期——当前阶段判断

    截至2026年6月，AI资本开支周期处于"扩张中期"（非末期）：

  • 美国云厂商层面：2025年Q1微软/谷歌/亚马逊/Meta合计资本开支约850亿美元，同比增长约50%。资本开支指引仍在上调。
  • 中国层面：字节跳动、阿里巴巴、腾讯2024-2025年AI资本开支合计约1500亿元人民币，占美国水平的约15%，仍有大幅追赶空间。
  • 但风险正在积累：每一代AI硬件（从H100到B200到下一代）性能大幅提升，但AI应用层面的商业化落地速度未跟上算力投入速度。 如果2026年下半年市场仍未看到"AI杀手级应用"（如全功能AI Agent、显著取代人类工作的AI产品），云厂商的资本开支节奏可能从"爆发式增长"转为"温和增长"——这将导致光模块（中际旭创、新易盛）、AI服务器（浪潮信息、工业富联）的需求增速放缓。

    • 风险量化：光模块订单增速每下降10个百分点，中际旭创估值压缩约20%。这是一个"增速敏感"而非"绝对增速敏感"的市场。

    主题四：人民币汇率——出海企业的隐形利润变量

    对于有美元收入的科技公司（中际旭创、新易盛、立讯精密、工业富联等），人民币贬值是一个显著的利润增厚因子。一个"粗略但有统计意义"的经验法则：

    人民币每贬值1%（对美元），有美元收入的科技公司的净利润增加约1.5-2.5%。

    原理：这些公司的美元收入不变，但折算为人民币后增加。同时，其成本（原材料、人力、水电）以人民币计价，不受汇率直接影响。收入端升值的"翻译效应"直接转化为利润增厚。

    2024-2026年，中国央行为了刺激经济增长（应对房地产市场下行），实施了较宽松的货币政策，人民币从6.8贬值至7.3区间。这一趋势如果延续，对上述企业形成持续利好。但如果人民币意外升值（如中美贸易协议推动），则是负面因素。

    主题五：地产与科技板块的"间接关联"

    科技板块与房地产行业的直接相关性趋近于零——AI芯片公司的收入不依赖于楼盘销售，光模块出口商不受首付比例影响。但"间接关联"确实存在，且不可忽视：

  • 地方政府补贴：许多半导体和新兴科技公司的扩张依赖地方政府补贴。地方政府的收入中，土地出让金（通过房地产开发商拍地）贡献约30-40%。2024年中国土地出让金同比下降约15-20%，地方政府财政吃紧，对科技企业的补贴力度面临下行风险。这一路径中受影响最直接的公司：中芯国际、沪硅产业、北方华创——它们的产能扩张需要政府资金配套。

  • 居民财富效应：中国居民家庭财富中约60-70%集中在房地产。房价下跌意味着居民信心下降、消费支出萎缩。这间接影响消费电子（立讯精密、京东方、瑞芯微）的终端需求。但这一传导路径较长、时滞约2-3个季度，且消费电子公司通常以全球市场为主，中国市场只是其中一部分。

  • 资金"挤出效应"：2020-2021年地产繁荣时期，大量资金从房地产流入股市（通过信托、理财产品、炒房者抛售房产进入股市），科技板块收益溢出。2022-2024年地产下行，部分资金流出股市进入存款或海外资产，科技板块失去了一部分"流动性溢价"。

    • 主题六：黄金相关性——科技股vs黄金的"跷跷板"

    科技成长股与黄金的历史相关性约为-0.3至-0.5（中等负相关）。这种负相关的驱动机制如下：

  • 科技股是"长期久期资产"——其当前估值中，约70-80%来自5年之后的远期现金流折现。这意味着科技股对实际利率极度敏感。
  • 黄金同样对实际利率敏感——但方向相反：实际利率上升，黄金持有成本增加，金价下跌；实际利率下降，黄金持有成本降低，金价上涨。
  • 实际利率≈名义利率−通胀预期。当实际利率上升时（如美联储加息），科技股和金价同时承压；当实际利率下降时（如美联储降息），两者同时受益。
  • 但某些场景下两者负相关特别突出："美联储宽松但经济不衰退"（如2024年软着陆预期）→科技股大涨+金价盘整/小跌；"经济恐慌"（如2025年某月美国衰退指标上升）→黄金大涨+科技股承压。

    • 总结： 科技板块与黄金的负相关性在中性宏观经济环境下约为-0.3至-0.4，但在极端尾部风险事件中（全球衰退、地缘冲突）负相关性会大幅增强，成为有效的"危机对冲组合"——但这不是一种稳定的套利关系。
