行业深度分析

40. 大秦铁路 (601006) — 中国"西煤东运"第一通道，PE:10.5, PB:0.6, ROE:1.44

因果链1：煤炭运量与宏观经济的直接映射 大秦铁路（大同-秦皇岛）是中国最主要的"西煤东运"专线铁路，年运量约4-4.2亿吨（占全国铁路煤运量的20%+） → 煤炭运量与火电发电量的相关系数约0.85 → 火电发电量=全社会用电量×(65-70%) → 全社会用电量增速取决于GDP增速（电力弹性系数约1.0-1.2） → 所以大秦铁路的运量本质上是"中国经济增长的影子指标"。若中国GDP从5%降至4% → 用电量增速从6%降至4% → 火电发电量从2%降至1% → 大秦线运量从4.2亿吨降至4.0亿吨 → 但铁路运价（吨公里约0.15元）由发改委核定（稳定性极高） → 量降但价稳 → 利润下降幅度相对有限。大秦铁路的利润稳定性在交运行业中名列前茅。

因果链2：蒙华铁路竞争与"运煤干线"生态变迁 蒙华铁路（浩吉铁路，2019年投运，年运力2亿吨）是直接挑战大秦铁路线路垄断地位的新竞争者→ 蒙华运煤从蒙古/陕北至华中（两湖一江）→ 但大秦的核心腹地是晋北/蒙西至环渤海港口（下水转海运至东部沿海电厂）→ 两者的服务港口/市场存在地理差异：大秦重点是沿海市场、蒙华重点内陆市场 → 不完全竞争：大秦的护城河来自港口基础设施（秦皇岛港/曹妃甸港的装船配套）→ 纯铁路竞争很难完全替代整个物流链。但蒙华的低价（运价较大秦低5-8元/吨）仍可能拉低大秦的新合约运价或降低其议价能力。

因果链3：高分红+PB破净的"格雷厄姆集结号" 大秦铁路持续分红25年（分红率约55-60%）→ 2024年股息率约5.5-6% → 叠加PB仅0.6倍（股价低于净资产40%） → 在格雷厄姆框架中属于"净流动资产价值型机会"（Net-Net的一种变体） → 意味着假设大秦铁路立即清算，股东可获得每股净资产的0.6倍 → 反映市场对煤炭运输长期结构性衰退的折价定价。这一折价的修复需要三个条件：①中国GDP增速稳定在4%以上（防止运量继续下滑）；②蒙华铁路未引发大规模价格战；③煤炭"保供"政策持续（确保量的基本盘）。这些条件在当前宏观环境下不是必然成立的。

格雷厄姆评价： PE 10.5<15（通过），PB 0.6<1（通过），股息率5-6%优秀。格雷厄姆打分：4/5（唯一的扣分项：长期煤炭运输需求的下行趋势无法回避）。

42. 中国建筑 (601668) — PE:3.51, PB:0.39, ROE:2.77，全球最大建筑商

因果链1：房地产下行对"房建业务"的直接冲击 中国建筑的房建业务（占收入约60%）涵盖住宅+商业+公共建筑 → 房地产新开工面积从2021年19亿平米降至2024年7亿平米（-63%） → 中国建筑的新签房建合同额从2021年约2.8万亿降至2024年约2.2万亿（-21%）→ 下降幅度小于地产新开工跌幅，因为基建/保障房/城市更新项目部分替代了商品房建设 → 但房建业务的毛利率（约7-8%）远低于基建（约10-12%）→ 收入结构向基建倾斜间接拉低了综合毛利率。核心关注指标：房屋新开工面积何时触底回稳（三年复合增速由负转正）是中国建筑利润复苏的必要条件。

因果链2：PPP项目投资的风险暴露——"隐形债务"的灰犀牛 中国建筑通过PPP/EPC模式参与基础设施投资 → 2020年前签约了大量PPP项目（约5000-6000亿元）→ 地方政府财政压力加大 → 部分PPP项目的回款周期从3年延长至5-7年 → 应收账款周转天数从60天延长至120-150天 → 应收账款减值风险积累 → 但中国建筑的客户主要是省/市级政府平台，实质违约风险较低（PPP项目多为有收益的经营性项目）→ 在报表上体现为大额合同资产（约3000-4000亿元）。投资者面临的挑战：无法区分"正常的期限延长"和"潜在的坏账风险"→ 这是中国建筑PE仅3.5倍的核心原因——市场对或有坏账风险的深度不确定性给予了极高的折价。

因果链3：极低PB——"折价来自资产风险vs市场悲观"的尖锐论战 PB 0.39 = 每1元中国建筑股票背后代表2.56元的净资产 → 市场以6折不到的价格交易该资产 → 对比其他中字头基建央企（中国中铁PB 0.31、中国铁建PB 0.31、中国交建PB 0.33、中国电建PB 0.57）——整个建筑央企板块PB都在0.3-0.6倍区间 → 暗示市场认为这些企业净资产的相当部分已被高风险应收账款/PPP资产/低效固定资产"污染"。即：如果剔除资产端的潜在损失，调整后的净资产可能只有账面的一半。这一"隐藏减值"（Hidden Impairment）究竟是多少？分析的典型做法：按3-5年应收账款账龄结构分析，以及PPP项目的内部收益率（IRR）是否低于融资成本。假设10-15%的净资产减值压力 → 调整PB约0.45-0.5倍，仍然极度便宜。

因果链4：国企价值重估（"中特估"）的政策催化 国家提出"中国特色估值体系"（2022年）→ 国资委要求央企提升ROE和估值（2024年新考核指标：增加市值管理权重）→ 中国建筑推出回购/增持/提高分红率的措施（2024年分红率从18%提升至25%）→ 股息率从2.5%提升至4.5-5% → 但仍远低于行业平均6-7% → 因为净利润被减值准备和折旧压低了。

格雷厄姆评价： PE 3.51（极低），PB 0.39（极低），有吸引力。但ROE 2.77%太差，且杠杆率（资产负债率约75%）太高，不符合格雷厄姆关于"保守财务结构"的要求。格雷厄姆打分：3/5（估值极端的吸引力vs.资产质量的不确定性打成平手）。

43-48. 其他建筑/工程央企

中国中铁 (601390) — 铁路+公路+城轨为主的基建巨头 新签合同额约3万亿（2024年）→ 铁路（25%）/公路（20%）/城轨（15%）/市政（25%）/房建（15%） → 与财政策略高度绑定：特别国债（超长期国债）发行规模从1万亿增至3万亿 → 基建投资资金来源充足 → 中国中铁的基建订单增速从2%提升至8-10%。但预付款比例下降导致垫资压力加大。格雷厄姆打分：3/5（PE+PB双低）。

中国铁建 (601186) — PE:4.89, PB:0.31, ROE:1.38 "一带一路"受益标的——中东/东南亚/非洲的铁路/公路基础设施EPC总包 → 海外收入占比约15-20%（在建筑央企中最高） → 沙特NEOM未来城/中泰铁路/印尼雅万高铁等大项目提供结构性增长。但海外EPC的毛利率（15%）高于国内（8%），但回款风险也大（欠发达国家的政府违约风险）。PB 0.31倍意味着市场认为其净资产中约70%存在潜在减值风险——这若属实，则是极度悲观假设（可能需要基建投资长期降速才能触发）。格雷厄姆打分：3/5。

中国交建 (601800) — 港口/桥梁/高速公路/疏浚 全球最大港口设计建设商 → 港珠澳大桥/深中通道的核心承建方 → 高速公路运营资产（持有约50条高速公路权益）→ 运营利润约占总利润的25-30% → 高速公路运营业务提供相对稳定的利润基础（经营性现金流可预测性强）→ 使中国交建的估值在建筑央企中相对有支撑。格雷厄姆打分：2/5。

中国电建 (601669) — PE:10.66, PB:0.57, ROE:1.31 水电/风电/光伏/抽水蓄能的设计施工一体化龙头 → 抽水蓄能是典型增量：中国规划2025年抽水蓄能装机6200万千瓦，2030年1.2亿千瓦，中国电建在该领域的市占率约80% → 将成为未来5年最确定的增长来源。但电力运营（持有约20GW的新能源电站）长期目标是轻资产运营转型，但短期仍以EPC工程为主。格雷厄姆打分：2/5。

中国中冶 (601618) — PE:8.46, PB:0.54, ROE:1.59 冶金工程（钢铁产能建设+钢厂改造）+ 矿产资源开发（镍矿/铜矿/锌矿） → 钢铁行业产能改造需求（向短流程电炉转型、碳减排技术改造）提供冶金工程的结构性需求 → 矿产资源（巴布亚新几内亚瑞木镍钴矿）为公司贡献约10%利润 → 镍价受锂电正极材料需求影响（高镍三元电池vs.磷酸铁锂的路线之争）。格雷厄姆打分：2/5。

中国能建 (601868) — PE:20.38, PB:0.99, ROE:1.25 电力工程（火力发电+新能源+电网设计施工） → 中国能建与中国电建是"电力系统双子星"（能建偏火电/送变电，电建偏水电/新能源）→ 2024-2026年火电建设回暖（因为电力保供压力，煤电新增核准加速）→ 中国能建的火电EPC业务短期受益。但PB 0.99为建筑央企中最高（市场最不悲观），对应的是其电力运营资产的更透明估值。格雷厄姆打分：2/5。
