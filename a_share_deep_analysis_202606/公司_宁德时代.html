    6. 宁德时代 (300750) — 全球动力电池龙头，CSI300第2权重（约4.20%），PE:22.51, PB:5.71, ROE:5.98

    因果链1：上游原材料价格暴跌带来利润率修复——但这是不可持续的"红利"吗？ 碳酸锂价格从2022年11月60万元/吨高点暴跌至2024年约10万元/吨 → 电池级碳酸锂每吨下降10万元 → 对应磷酸铁锂(LFP)电芯成本下降约0.05元/Wh → 宁德时代2024年出货量约560GWh → 成本下降释放约280亿元利润空间。但问题是：下游整车厂也要求同步降价 → 宁德时代的最终利润率改善幅度取决于其议价能力（即"降价幅度在产业链中如何分账"）。2024年实际结果是：宁德时代毛利率从22%提升至26%左右，大约60%的成本下降留在了自己的利润表、40%让渡给了客户。这一分账比例反映了宁德时代的强大议价权（cr4 > 50%的集中度优势）。但若下游需求进一步走弱，整车厂可能要求更大比例的降价让利。因果链2：全球化受阻——美国市场的事实性封锁 US IRA法案（2022年8月签署）→ 每辆电动车补贴$7500，但电池组件须在北美制造或与FTA国家采购 → 宁德时代以LFP技术优势寻求曲线进入 → 以"技术授权+工厂运营"模式与福特在密歇根合作（不直接持股，规避中国FEOC限制）→ 但2024年美方加强对FEOC定义的严格解释，扩展至"适用技术许可" → 宁德时代的美国战略有效受阻。影响：美国电动车市场（2024年约1200万辆/年，2030年预计3000万辆）对宁德时代几乎零直接敞口 → 宁德时代的成长上限被结构性压缩。作为对冲，宁德加大向欧洲/东南亚的产能布局。因果链3：欧洲CBAM与碳关税的非关税壁垒 欧盟碳边境调节机制(CBAM)于2026年正式征收 → 宁德时代出口欧洲电池需要购买碳配额（约80-100欧元/吨CO2）→ 每kWh电池的碳排放约60-80kg CO2 → 增加碳税约5-8欧元/kWh → 而宁德时代在欧盟（德国图林根/匈牙利德布勒森）本地工厂的碳排放更低（欧洲电网更清洁，碳密集度约0.25吨CO2/MWh vs 中国约0.55吨CO2/MWh）→ 因此欧盟CBAM反而加速宁德在欧洲本地建厂的ROI——本地工厂的碳成本优势≈5-8欧元/kWh的价差，足够覆盖欧洲人力成本高的劣势（欧洲人工约40欧元/小时 vs 中国10欧元/小时）。这是一个"看似利空实则因子加速"的复杂传导。因果链4：储能"第二曲线"打开估值空间 大型储能系统(ESS)的需求爆发（2024年中国储能装机约80GWh，预计2030年500GWh+）→ 宁德时代储能电池出货占比从2022年12%提升至2025年约25% → 储能电池与动力电池共享产线（可根据两类订单的价格动态分配产能）→ 当动力电池需求增速放缓时（EV渗透率从50%到70%的S曲线后半段），储能提供增量。更为关键的是，储能客户的付款条件通常优于整车厂（回款周期60天 vs 90-120天），有助于宁德时代改善经营性现金流（OCF）。因果链5：利率路径对成长股估值的双重影响 Fed加息周期（2022-2023）→ 无风险利率从0.25%升至5.5% → DCF估值模型中的折现率从7%升至10%+ → 远期现金流（5-10年后的储能/海外利润）现值缩水约40-50% → PE压缩至20倍区间。若Fed降息（降息周期开启）→ 折现率下降 → 远期现金流现值回升 → PE中枢从20倍重新升至30-35倍。但对于宁德时代而言，降息对成长股估值的提振可能被贸易摩擦加剧（关税）部分抵消——这是2025-2026年投资者需要权衡的核心矛盾。黄金相关性： 负相关。避险情绪高时（黄金涨）→ 资金从成长股流向避险资产 → 宁德时代承压。历史相关约-0.2至-0.3。

    7. 比亚迪 (002594) — 新能源车+电池一体化，PE:50.02, PB:3.52, ROE:1.65

    因果链3：锂价下跌对一体化利润结构的影响 碳酸锂价格从60万元/吨跌至10万元/吨 → 对宁德时代而言是正面（降低原料采购成本），对比亚迪而言则是双刃剑 → 正面：比亚迪自产电池（弗迪电池）的原材料成本同步下降 → 每辆车电池成本下降2000-3000元 → 但负面：比亚迪在2021-2022年高价期大量屯了锂盐库存（约8万吨碳酸锂当量→按60万元/吨的成本计算，这部分库存从"隐形收益"变成"浮动亏损" → 库存减值需在报表中反映。更微妙的是，比亚迪的整车利润来源中，电池外供比例仍不到10%，所以锂价下降对电池利润的直接增厚有限——比亚迪受益更多来自消费者信心（低价车刺激需求）而非成本改善本身。
