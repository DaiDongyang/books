5. 上汽集团（600104）—— PE:10.11, PB:0.41, ROE:1.01%

上汽集团的核心利润来源长期以来是旗下合资公司（上汽大众、上汽通用、上汽通用五菱）的权益法投资收益，历史上贡献集团净利润的60-80%。传导路径为：德系（大众）和美系（通用）品牌在中国市场面临自主品牌（比亚迪、吉利、奇瑞）的激烈竞争 → 上汽大众年销量从2019年的200万辆降至2025年的约100万辆 → 上汽通用年销量从167万辆降至约70万辆 → JV净利润从峰值约400亿元下降到2025年的约50亿元 → 集团合并净利润从峰值约300亿元降至约80-100亿元 → 分红能力下降（股息率从约5%降至约2-3%）。这一链条目前尚未见底，德系/美系在中国市场的份额从2020年的约25%降至2026年的约15%，仍在下降通道。2024-2026年汽车行业价格战向上汽等合资车企的冲击尤为猛烈。传导路径为：特斯拉/比亚迪带头降价 → 合资车企被迫跟进 → 终端优惠幅度扩大至历史最高（部分车型折扣率达20-30%）→ JV的出厂价（transfer price）被压缩 → 合资企业毛利率从约18%降至约10% → JV净利润断崖式下跌。格雷厄姆角度看，上汽PE 10倍、PB仅0.41倍看似极度低估，但ROE仅1%意味着盈利正在接近盈亏平衡点——这是典型的"价值陷阱"特征，需要等待JV利润企稳的信号。

26. 比亚迪 (002594) -- 估值扩张后的压力测试

中国新能源汽车行业在2023-2026年进入残酷的"淘汰赛"阶段。比亚迪2025年新能源汽车销量约400万辆，但净利润却同比下降19%，核心原因在于价格战。因果链为：更多车企进入新能源市场（小米、华为等） → 供给过剩 → 价格战加剧 → 比亚迪主力车型（秦、宋、汉、海豚等）多次降价 → 单车净利润从约1万元降至约6,000元 → 即使销量增长，净利润仍下降。比亚迪正在全球布局产能以规避贸易壁垒。欧盟对中国电动车加征反补贴关税（2024年落地），美国维持27.5%的进口关税。因果链为：海外关税壁垒 → 中国整车直接出口利润率下降 → 海外本地化建厂 → 资本支出大幅增加（单个工厂投资约30-50亿元） → 短期折旧和运营成本上升 → 海外工厂需要3-5年才能实现盈利贡献。这一长期逻辑在短期内是利润压力的来源。因果链三：锂电池原材料成本下降 -> 刀片电池成本优势 -> 比亚迪垂直一体化护城河比亚迪与大多数车企的核心区别在于垂直一体化——从电池、电机、电控到IGBT芯片、车规级半导体，比亚迪掌握新能源汽车的完整产业链。碳酸锂价格从2022年高点的约60万元/吨跌至2025年的约8万元/吨，使电池成本下降了约70%。对于比亚迪而言，其自研刀片电池的成本（约0.4元/Wh）已显著低于外购电池的成本（约0.6元/Wh），这一成本优势在价格战中的价值被放大：竞争对手在亏本卖车时，比亚迪仍有微利。格雷厄姆评估: PE 29.7倍和PB 3.52倍均超标。高ROE（20.8%）反映的是优秀的产品竞争力，但当前PE隐含的增长预期在价格战中面临挑战。比亚迪的垂直一体化优势是一柄双刃剑——在行业上行期带来超额利润，在行业下行期也意味着更大的固定成本压力。评分: 1/5。

7. 比亚迪 (002594) — 新能源车+电池一体化，PE:50.02, PB:3.52, ROE:1.65

因果链1：终端价格战——这是生存战还是份额战？ 2023年特斯拉率先降价（Model 3从23万降至19万）→ 比亚迪被迫跟进（秦Plus DM-i从15万降至10万以内，降幅33%）→ 单车ASP从18万下降至14万元 → 而单车BOM成本降幅仅约2-3万元（电池成本下降+规模效应） → 单车毛利从4万缩至2.5万 → 净利润/辆从约1万降至约4000-5000元。传XX径为：降价→销量增长但利润缩水→必须依赖更大的销量规模来维持总利润→这逼迫比亚迪进一步降价→进入"降价-增量-再降价"循环，直到行业中边际产能（如哪吒/零跑/合众的出清）。行业出清拐点的标志：CR3（比亚迪+特斯拉+某个头部企业）市占率超过65%。因果链2：欧盟反补贴关税——区域市场利润的被稀释 欧盟对华电动车反补贴调查（2023年10月启动）→ 终裁决定对比亚迪（唯一配合调查的企业）征收17%额外关税，对其他企业征收17-36% → 比亚迪元Plus(Atto 3)在欧洲售价约3.5万欧元 → 加征17%关税后终端价格升至4.1万欧元 → 与欧洲本土竞品（大众ID.4约4万欧元、标致e-2008约3.8万欧元）的价差优势从10%转为0 → 但比亚迪选择在匈牙利（欧盟成员）建厂（2025年投产，年产15万辆）→ 本地生产绕过关税壁垒 → 但匈牙利工厂初始产能爬坡期（1-2年）的成本高于国内出口 → 欧洲业务的利润率从8-10%（出口）降至3-5%（本地化初期），要在2027年才能恢复至正常水平。因果链3：锂价下跌对一体化利润结构的影响 碳酸锂价格从60万元/吨跌至10万元/吨 → 对宁德时代而言是正面（降低原料采购成本），对比亚迪而言则是双刃剑 → 正面：比亚迪自产电池（弗迪电池）的原材料成本同步下降 → 每辆车电池成本下降2000-3000元 → 但负面：比亚迪在2021-2022年高价期大量屯了锂盐库存（约8万吨碳酸锂当量→按60万元/吨的成本计算，这部分库存从"隐形收益"变成"浮动亏损" → 库存减值需在报表中反映。更微妙的是，比亚迪的整车利润来源中，电池外供比例仍不到10%，所以锂价下降对电池利润的直接增厚有限——比亚迪受益更多来自消费者信心（低价车刺激需求）而非成本改善本身。因果链4：智能化与自动驾驶的"投资悖论" 比亚迪宣布2024年起全系标配智驾系统（DiPilot+高阶智驾包）→ 每辆车增加1000-2000元BOM成本（激光雷达+摄像头+域控制+算法投入）→ 目前消费者对智驾的支付意愿约为3000-5000元（调研数据）→ 短期来看智驾投资是"亏本买卖"（收入<成本）→ 但这是必要的防御性投资——如果不做智驾，比亚迪在15万以上市场的份额将被小鹏/蔚来/华为系持续蚕食。这个"成本先行、收益在后"的结构，将压制比亚迪短期利润2-3个季度，但可能成为2027年后的竞争壁垒。格雷厄姆评价： PE 50倍（极端高估），PB 3.52（高估），ROE 1.65%（极低，反映2024年盈利低谷）。完全不符合。打分：0/5。对格雷厄姆派投资者来说，比亚迪是可观测、不可投资的对象——需要等待利润释放后的PE压缩。

27. 上汽集团 (600104) -- 格雷厄姆悖论的终极案例

上汽集团长期依赖上汽大众、上汽通用两家合资企业的利润贡献。2020-2025年，合资品牌在中国市场遭遇自主品牌（比亚迪、吉利等）和新势力的猛烈冲击，销量和利润率持续下降。因果链为：合资品牌燃油车销量萎缩（大众在中国从2020年约380万辆降至2025年约200万辆） → 合资企业净利润大幅下降 → 上汽集团（50%持股）享有的投资收益减少 → 上汽整体净利润从2020年约200亿元降至2025年约30-40亿元 → 净资产（PB仅为0.41倍）几乎不产生回报（ROE仅1%）。上汽集团并非没有意识到电动化转型的必要性。公司先后推出了飞凡（R品牌）和智己（L品牌）两个独立新能源品牌，试图复制比亚迪的成功路径。但这两个品牌的表现远低于预期：2025年智己月销量不足5,000辆，飞凡月销量不足3,000辆，合计不到比亚迪月销量的5%。因果链为：新能源转型战略启动，品牌定位不够清晰（智己与飞凡相互竞争），产品定价缺乏竞争力（与蔚来、理想、小鹏同价位但品牌力不足），渠道建设缓慢，销量持续低迷，研发投入和市场营销费用却持续消耗，自主新能源板块仍处于巨额亏损状态，无法对冲合资品牌利润的下滑。

4. 比亚迪（002594）—— PE:50.02（港股口径约30x）, PB:3.52, ROE:20.8%（更正：用户给的ROE 1.65可能为单季数据，年报ROE实际约20.8%）

中国新能源乘用车渗透率从2020年的约6%飙升至2026年6月的约55%以上，比亚迪在其中占据了约35%的市场份额。传导路径为：双碳政策+新能源补贴（历史）+车辆购置税减免 → 消费者购买新能源倾向上升 → 比亚迪凭借DM-i混动+纯电全产品线抢占市场份额 → 年产销量从2020年的42万辆暴增至2025年的约500万辆 → 规模效应使每辆车摊销的研发/模具/管理费用大幅下降 → 单车成本下降20-30% → 即便在价格战环境中仍能维持约4-5%的汽车业务净利率。这一链条的尾部风险在于：如果渗透率超过70%后增速放缓，比亚迪销量增速从50%+降至10-15%，规模效应对利润的边际贡献将递减。比亚迪2025年海外销量约80万辆（占总销量约16%），主要市场包括东南亚、欧洲、南美。传导路径为：欧美对中国电动车加征关税（欧盟约17-25%、美国约100%）→ 整车出口成本大幅上升 → 倒逼海外建厂 → 比亚迪在泰国、匈牙利、巴西、印度尼西亚等地建设工厂 → 海外产能2026-2028年逐步释放 → 海外市场毛利率（约20-25%）高于国内（约15-18%）→ 提升整体利润率。但海外建厂面临地缘政治风险：匈牙利工厂可能受中欧关系波动影响，巴西工厂面临南美政策不稳定性。碳酸锂价格从2022年的60万元/吨跌至2026年的约8-10万元/吨。传导路径为：锂价暴跌 → 电池级碳酸锂成本下降 → 比亚迪垂直整合（弗迪电池自供）使电池成本下降约30-40% → 单车电池成本节省约5000-8000元 → 比亚迪选择降价保份额（2024-2025年秦/宋/汉等车型数次降价）→ 以价换量维持35%市占率 → 虽然单车利润从约1万元降至约6000-7000元，但总量增长支撑净利润。如果锂价反弹（如南美锂矿供应受地缘影响或电动车需求超预期），比亚迪的电池成本优势将被削弱。

6. 长城汽车（601633）—— PE约25-30x（周期高点的失真PE）, PB约2.0-2.5x, ROE约8-10%

长城在新能源转型上总体落后于比亚迪。传导路径为：新能源积分政策要求 → 长城新能源车（欧拉/魏牌）销量占比仅约25%（远低于行业55%的渗透率）→ 需要购买新能源积分（每个积分约500-800元）→ 增加营业外支出约10-20亿元/年。如果2026-2027年新能源渗透率进一步提升，长城的双积分压力将持续加大，侵蚀燃油车业务的利润。

7. 广汽集团（601238）—— PE:-22.8（亏损）, PB:0.57, ROE:-0.63%

广汽集团的核心盈利曾经完全依赖广汽丰田和广汽本田两家日系合资公司。传导路径为：日系车在中国市场份额从2020年的约24%暴跌至2026年的约10% → 广汽丰田年销量从100万辆降至约50万辆 → 广汽本田从80万辆降至约40万辆 → JV权益利润从峰值约150亿元降至2025年的约30亿元 → 2026年可能进一步降至接近零。原因包括：（1）日系在电动化转型极度缓慢，（2）中国消费者对日系品牌的偏好下降（地缘政治+产品力相对落后），（3）日系混动技术（THS/i-MMD）被比亚迪DM-i超越。这一链条目前没有见底的信号——日系尚未推出有竞争力的纯电产品。传祺（燃油+混动）和埃安（纯电）是广汽自主品牌的双轮。埃安2025年销量约50万辆，在纯电市场排名第三（仅次于比亚迪、特斯拉）。传导路径为：埃安在中低端纯电市场（AION Y/S/V）占有一定份额 → 但面临比亚迪元/秦Plus EV和零跑C10的激烈价格竞争 → 埃安单车净亏损约3000-5000元（扣除补贴后）→ 虽然亏损幅度较2023年（单车亏约10000元）明显收窄，但尚未扭亏 → 自主板块合计亏损约30-50亿元/年 → 加上JV利润下降 → 集团合并层面出现净亏损。广汽的出路在于埃安高端品牌（Hyper昊铂）能否在20万以上市场打开局面。

13. 华域汽车（600741）—— PE约8-10x, PB约0.8-1.0x, ROE约6-8%

华域正在加大对外部客户（包括特斯拉、比亚迪、蔚来、理想等）的配套比例。传导路径为：意识到对上汽的单一依赖风险 → 开拓非上汽客户 → 外服客户收入占比从2022年的约30%提升至2025年的约40% → 但外服客户的配套需要前期研发投入（模具/产线投资）→ 短期内拖累利润但长期价值体现在客户多元化。这一链条的瓶颈在于：华域作为上汽控股子公司，其他整车厂（特别是比亚迪等竞争对手）在采购时存在顾虑（担心技术和成本信息泄露给上汽）。
