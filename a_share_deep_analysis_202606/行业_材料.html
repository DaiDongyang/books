    • 45. 金山办公（688111）—— PE约50-70x, PB约10-15x, ROE约10-12%

    金山办公（WPS Office）是中国最大的办公软件公司，核心产品WPS Office在国内市占率约40-50%（仅次于Microsoft Office）。传导路径为：信创政策要求党政机关和央国企采购国产办公软件 → WPS在信创市场中占据约90%以上份额（微软因制裁无法进入信创市场）→ 信创带动的WPS机构授权收入从2020年的约10亿增至2025年的约30-40亿 → 占WPS总收入的约30-35%。这一信创红利预计持续到2027-2028年（信创替换高峰）。

    WPS的个人用户基础约4-5亿月活，但付费率仅约5-6%（约2000-3000万付费用户），相比Microsoft 365的约15-20%付费率有显著提升空间。传导路径为：WPS从免费+广告模式转向订阅（会员/WPS会员）→ 广告收入下降（用户反感广告）→ 会员收入增长 → WPS会员年费约89-198元/年 → 每提升1%的付费转化率意味着约6-8亿增量收入。AI功能（WPS AI，智能写作、PPT生成）是拉动付费转化率提升的关键：WPS AI会员定价高于普通会员（约300元/年），且吸引了对AI办公工具感兴趣的新用户。

    金山办公通过WPS AI（基于大模型的智能文档、表格、PPT生成功能）大幅提升产品力。传导路径为：AI办公功能（一键生成PPT、AI写作、智能数据处理）→ 提升WPS在"易用性"上对Microsoft Office的相对竞争力 → 吸引更多企业付费用户（特别是中小企业和个人专业用户）→ 缩小与微软Office在功能体验上的差距 → 可能的国际化扩张（WPS在东南亚/非洲/拉美的用户基础持续扩大）。PE 50-70倍的高估值主要源自市场对WPS AI大规模用户基础变现和国际化潜力的长期定价，但高PB（10-15x）在格雷厄姆框架下属于极度高估。

    中金公司（601995）

    中金公司是中国历史最悠久、最国际化的投资银行，PE约11.6倍，PB约1.6倍，ROE约3.4%（TTM，年化约6-8%）。

    中金公司是中国投行费率最高的券商，专注于大型国企、央企和新经济龙头企业的IPO、配股、并购等高端投行业务。单笔IPO的保荐承销费可能高达1-5亿元。当IPO市场开放度提高（如大型央企回A上市、大型科技公司港股IPO），中金公司是最大的受益者之一。但当IPO收紧，投行收入锐减的速度也快于同行，因为中金的小项目储备相对较少。

    中金公司是中国券商中跨境业务能力最强的，设有香港、纽约、伦敦、新加坡等分支。当中国企业赴海外上市的通道畅通（如中美审计监管合作稳定），中金公司的国际投行收入约占总收入的20-30%，远高于同行。但当中美关系紧张导致中概股退市风险上升，国际业务收入受直接影响。

    中金财富的代销和投顾业务定位高净值人群（500万以上），AUM约5000-6000亿元。服务高净值客户的费率较高（管理费约0.5-1%），因此中金财富的人均创收能力远高于同行的零售业务。高净值客户的资产配置相对稳定（在市场下跌时也不会大量赎回），因此收入稳定性优于同行。

    格雷厄姆评分：PE约11.6 < 15（得1分），PB约1.6 > 1.5（得0分），股息率约1.5-2%（得0分），连续盈利20年以上（得1分），ROE约6-8%（得1分）。综合评分：3/5。

    主题六：黄金相关性——科技股vs黄金的"跷跷板"

  • 科技股是"长期久期资产"——其当前估值中，约70-80%来自5年之后的远期现金流折现。这意味着科技股对实际利率极度敏感。
  • 黄金同样对实际利率敏感——但方向相反：实际利率上升，黄金持有成本增加，金价下跌；实际利率下降，黄金持有成本降低，金价上涨。
  • 实际利率≈名义利率−通胀预期。当实际利率上升时（如美联储加息），科技股和金价同时承压；当实际利率下降时（如美联储降息），两者同时受益。
  • 但某些场景下两者负相关特别突出："美联储宽松但经济不衰退"（如2024年软着陆预期）→科技股大涨+金价盘整/小跌；"经济恐慌"（如2025年某月美国衰退指标上升）→黄金大涨+科技股承压。

    • 总结： 科技板块与黄金的负相关性在中性宏观经济环境下约为-0.3至-0.4，但在极端尾部风险事件中（全球衰退、地缘冲突）负相关性会大幅增强，成为有效的"危机对冲组合"——但这不是一种稳定的套利关系。

    1. 贵州茅台 (600519) -- "液体黄金"的周期之痛

    估值现状: PE 22.3倍（TTM），PB 7.09倍，ROE 32.53%，2025年营收1,688亿元（-1.21%），净利润823亿元（-4.53%），为上市以来首次营收下滑。

    贵州茅台的销售结构中，政务与商务宴请占比历来显著。中国房地产市场自2021年见顶后持续下行，2025年全国商品房销售面积约9.5亿平方米，较高峰期（2021年约18亿平方米）缩水近半。这一变化的宏观传导机制如下：房地产销售萎缩 → 地方政府土地出让金收入从2021年约8.7万亿元骤降至2025年约4万亿元 → 地方财政进入紧缩周期 → 各级政府和事业单位的接待经费被大幅压缩 → 以茅台为代表的高端白酒政务消费场景减少。这一链条在2024-2025年尤为明显，地方财政"过紧日子"政策使茅台在公务宴请中的出现频率显著下降。虽然茅台近年大力拓展直销渠道（i茅台平台），但该平台的营收在2025年同比下降34.9%至130亿元，反映批价倒挂环境下消费者通过官方平台购买的动力减弱，也说明地产财政收缩对消费需求的传导已经扩散到零售终端。

    中国消费者信心指数在2024-2025年持续处于历史低位（约85-88点，远低于100的荣枯线），这一宏观情绪的变化通过"礼品消费"管道直接传导至茅台。茅台作为中国商务礼品市场的"硬通货"，其需求与企业和个人的送礼意愿高度正相关。消费者信心下降 → 企业缩减商务送礼预算 → 飞天茅台礼品需求走弱 → 批发价从2024年约2,400元/瓶跌至2025年低点约1,600元/瓶 → 经销商（黄牛）持有库存的预期收益转负 → 抛售库存加速价格下跌 → 倒挂压力下经销商减少回款 → 茅台被迫在2025年Q4主动停止向渠道发货。这一链条在2025年呈现典型的"库存螺旋"：茅台2025年Q4营收同比暴跌19.35%，净利润暴跌30.34%，本质上不是终端消费突然消失，而是渠道库存从"正向补库"转向"负向去库"带来的需求真空。2026年Q1茅台营收恢复增长+6.34%，净利润+1.47%，说明去库存最剧烈的阶段已经过去。

    人民币兑美元在2024-2026年经历阶段性贬值（从约7.0贬至约7.3），理论上使中国出口品的美元标价更具竞争力。茅台海外收入约48.5亿元（仅占营收约2.9%），海外销售以华人聚居区和亚洲市场为主，欧美主流消费群体对白酒的接受度有限。即使人民币贬值带来约6-7%的海外定价优势，由于市场渗透天花板和品类认知限制，这一因子对茅台的边际影响可以忽略不计。

    中国货币政策在2024-2026年处于宽松周期，LPR从4.2%降至约3.0%，银行理财收益率从3.5%降至约2.0%，十年期国债收益率跌破2.0%。在这一环境下，茅台老酒和生肖酒的"收藏投资"属性被放大——当传统金融资产的预期回报率下降，茅台被称为"液体黄金"的叙事得到强化。但这一效应在2025年明显减弱：随着茅台批价从2,400元跌至1,600元，收藏投资的"赚钱记忆"被打破，老酒价格指数出现历史性下跌，投机性需求反成抛售压力。只有当批价企稳且其他资产收益率持续低迷时，这一因果链才会重新发挥作用。

    格雷厄姆评估: PE 22.3倍超过15倍上限，PB 7.09倍远超1.5倍安全线。作为超级品牌消费品，茅台的高估值反映的是品牌护城河和确定性溢价，而非格雷厄姆式"廉价股"特征。评分: 1/5。

    资产关联: 与黄金存在弱正相关（均为"存储价值"叙事），与房地产存在间接相关（通过财政效应和财富效应），与比特币无直接关联。

    2. 中国石化 (600028) — PE:8.84, PB:0.73, ROE:2.04，下游炼化+化工龙头

    因果链1：原油价格与炼化毛利率的反直觉关系（核心变量） 这是投资者最常误解的因果链。中石化是炼化为主（占利润70%）而非石油开采：国际油价下跌（布伦特从90跌至70美元）→ 炼厂原料（原油）成本同步下降 → 但成品油价格受发改委调价机制保护（40美元地板价、130美元天花板机制，中间区间按22个工作日平均变化调整）→ 成品油价格下降幅度通常小于原油成本下降幅度 → 裂解价差（Crack Spread）扩大 → 炼化毛利提升。因此，原油价格下跌反而利好中石化，与中石油完全相反。——这一关键差异是"多空配对交易"（做多中石化+做空中石油对冲油价风险）的基础逻辑。

    因果链2：化工周期与产能投放的致命交互 中石化化工板块（乙烯、丙烯、PX、PTA、聚烯烃）占利润约15-20% → 2019-2022年全球大化工产能扩张潮（中国民营大炼化+美国页岩气乙烷裂解+中东低成本产能）→ 化工品价格从历史中枢下跌30-40% → 聚烯烃-石脑油价差从500美元/吨收窄至200-300美元/吨 → 中石化化工板块从盈利50亿转为亏损约100亿元 → 拖累整体净利润。该周期底部何时结束？需观察民营大炼化产能利用率开始下降、关停高成本产能。这对中石化化工的转折点极为关键。

    因果链3：电动车渗透率与成品油需求峰值 中国新能源车渗透率从2020年5%升至2026年预计55%+ → 每百万辆燃油车替代 = 减少约300万吨成品油消费 → 中石化国内成品油销量2022年已达峰（约1.8亿吨）→ 此后每年递减2-4% → 加油站网络资产价值面临重估（加盟商投诉销量下降）。中石化的应对：加油站→加能站转型（增加充电桩/换电/氢能），但充电桩坪效（每个充电桩日周转率约3-4次）远低于加油站（日均加油量15-20吨），单位土地面积的能源销售价值下降60-70%。

    因果链4：USD/CNY的双向影响 美元升值（CNY贬值）→ 中石化进口原油人民币成本上升（尽管以美元结算的原油在全球市场降价，但换汇成本提高可能抵消部分收益）→ 炼化理论原料成本变动被汇率对冲。另一方面，中石化出口化工品（PX、MEG等）→ 人民币贬值使出口产品更便宜 → 出口竞争力提升 → 外销量增加。净效果取决于进口原油规模（约2.5亿吨/年）vs 出口化工品规模的比值（约500万吨），进口端影响远大于出口端，因此人民币贬值对中石化净利空。

    格雷厄姆评价： PE 8.84＜15（通过），PB 0.73＜1.5（通过），ROE 2.04%偏低但反映周期性低谷而非结构性亏损。股息率约5%。格雷厄姆打分：4/5。问题在于长期成长性：电动车替代将结构性侵蚀需求，化工产能过剩周期尚未出清。

    21. 紫金矿业 (601899) — 铜金双轮驱动，CSI300第6权重，PE~15-18, PB~3-4, ROE~10-15

    因果链1：Fed降息周期下的黄金+铜双受益——紫金独特的"对冲优势" 美联储开始降息（2024年9月首次降息50bp，2025-2026年预计再降150-200bp）→ 实际利率（名义利率减通胀预期）下行 → 持有黄金的机会成本降低 → 黄金ETF净流入回升 → 金价突破2500美元/盎司并向3000美元迈进 → 紫金矿业黄金业务（约占总毛利25-30%）受益。与此同时，降息刺激经济复苏预期 → 全球制造业PMI回升至50以上 → 铜的新增需求（电网/新能源/数据中心）同比增长3-5% → 铜供需从过剩（2024年约20万吨）转为缺口（2025年约10万吨） → LME铜价从8500美元/吨涨至10500美元/吨 → 紫金铜业务（约占总毛利40-45%）毛利率从45%扩大至55%+ → 金和铜的双重上行进一步被人民币贬值（USD/CNY从6.9走弱至7.3）放大（海外资产以美元计价、国内以人民币折算→汇兑收益增加）。紫金的独特之处在于：在通胀环境中（金涨）、在增长环境中（铜涨）、在衰退环境中（美元走弱→金涨） ——三个宏观经济场景中至少有两个对其有利，属于"反脆弱"资产。

    因果链2：铜矿"供给悬崖"的长期结构性逻辑 全球铜矿资本支出从2015年约1000亿美元降至2023年约650亿美元（长期投资不足）→ 现有大型铜矿（必和必拓Escondida、Freeport Grasberg、智利Codelco）的矿石品位从0.8%持续下降至0.4-0.5% → 每吨铜的现金成本上升15-20% → 新矿建设周期5-8年（2015年低谷资本开支=2023年后才有新矿投产）→ 形成"2025-2030年铜供给刚性缺口" → 市场对这一缺口的定价（forward curve从contango转为backwardation）→ 紫金矿业现有矿山（卡莫阿-卡库拉DRC、巨龙铜业西藏）的现金成本在4000-5500美元/吨 → 在铜价8000-10000美元/吨的环境中，每吨净利润高达3500-4500美元 → 卡莫阿二期（年产能45万吨）、三期（年产能增至60万吨）逐步达产 → 紫金2024-2028年铜产量CAGR约15% → 成为全球前五大铜生产商。这是紫金矿业估值故事中最扎实的环节——全球铜供给拐点的确定性极高（通过3-5年产能爬坡可验证）。

    因果链3：刚果金（DRC）和塞尔维亚的地缘政治风险——"高回报区域"的三重考量 紫金的核心增量资产卡莫阿-卡库拉铜矿位于刚果金（DRC）：东部的武装叛军（M23运动）活跃 → 但卡莫阿在南部科卢韦齐，距离冲突区域800km+,安全风险有限 → 但刚果金政府的政策风险值得关注：2023年刚果金审查采矿合同（FMI/FQM合同重新谈判）→ 虽然紫金的合同（按照卢本巴希公约条款）相对稳健，但"资源民族主义"的尾部风险不能被忽略。塞尔维亚的佩吉铜金矿：塞国政府环保压力 → 2022年曾吊销力拓的贾达尔锂项目许可 → 同理佩吉项目面临环保组织抗议 → 扩展产能审批进度可能滞后。这些地缘政治风险在牛市中被忽视、但在市场下跌时会成为显著的估值折价因子——紫金历史上PE波动区间为8-25倍，地缘政治是估值弹性的主要抑制因素之一。

    因果链4：人民币汇率波动的"镜像"效应 USD/CNY从6.3升至7.3（人民币贬值约15%） → 紫金矿业海外资产（卡莫阿卡库拉、塞尔维亚博尔、哥伦比亚武里蒂卡金矿）的美元现金流以人民币折算时直接增加15% → 同时，紫金的美元计价债务（约50亿美元）的折算减值效果（负债端缩水）轻微抵消该收益 → 但净效果为正。汇率对紫金的独特意义：它是一个"天然的人民币贬值对冲标的"，在人民币下行周期中，持有紫金的资产价值被动膨胀，这一逻辑与纯粹的黄金投资（通过黄金ETF）或铜期货投资（通过LME）完全不同。

    格雷厄姆评价： PE 15-18倍在周期顶峰偏高、周期谷底合理，PB 3-4倍超过1.5倍标准。ROE 10-15%在A股中属于中上。打分：2/5。格雷厄姆不会喜欢"资源周期股+海外政治风险"的商业模式。

    黄金相关性： 强正相关（0.6-0.7）。紫金的黄金业务占比约25-30%，叠加铜的金价协同效应（金铜矿伴生的成本优势），使紫金成为沪港通中"黄金除ETF外最佳的A股代表"。

    房地产相关性： 正向。房地产/基建用铜约占中国铜消费的20-25%，房地产投资每下降10%，铜需求下降约2-3%。但新能源车+数据中心+电网建设正在稀释这种关联：在2025年，铜消费中房地产相关的占比已从2019年的30%降至20%，趋势持续降低。

    23-31. 其他材料/矿业公司

    宝钢股份 (600019) — PE:13.78, PB:0.55, ROE:1.02 中国最大钢铁企业（产能约5000万吨/年）→ 钢铁是"宏观经济的晴雨表"——当房地产减速（用钢量下降约40-50%）+ 基础设施增速放缓 + 制造业用钢（汽车/家电）增长不足以对冲 → 热轧卷板（HRC）价格从2021年6000元/吨降至2024年3800元/吨 → 全行业进入微利/亏损状态。宝钢的特殊之处：高端取向硅钢（变压器用）和汽车板（高强钢/镀锌板）占比较高的产品结构 → 整体毛利率比行业平均高3-5个百分点 → 在周期低谷中仍能维持正利润。因果链：电网投资（特高压/配网升级）→ 取向硅钢需求年增10-15% → 宝钢取向硅钢产能约100万吨/年（占国内约60%）→ 单吨利润约3000-4000元/吨 → 是宝钢利润的"压舱石"。格雷厄姆打分：3/5（PE<15 OK，PB0.55极低，但ROE太差）。

    海螺水泥 (600585) — PE:16.45, PB:0.50, ROE:0.76 水泥是"房地产投资的最直接传感器"——房地产新开工（2021年19亿平米→2024年7亿平米，-63%）→ 水泥需求从24亿吨降至17-18亿吨（-25~30%）→ 海螺水泥的产能利用率从85%降至65% → 单位产品固定成本（折旧+人工）从50元/吨摊升至65元/吨 → 净利率从18%降至3% → 但海螺水泥的护城河在于：吨成本约为同行最低（石灰石矿自给率高+长江流域水运优势+非自有矿山的低勘探成本）→ 在行业亏损线（水泥价格300元/吨平衡线）附近时海螺仍有35-40元/吨的利润 → 使海螺在周期底部仍能保持净现金流正数，而半数同行（尤其是中小粉磨站）已在现金亏损状态下运营。等待行业协同减产（主动关停20-30%的产能）是水泥股利润复苏的关键条件，但这一过程在地方财政保护的干扰下可能需3-5年。格雷厄姆打分：3/5（PB 0.5极度便宜，PE 16.4略高但反映的是周期的底部利润）。

    中国巨石 (600176) — PE:39.21, PB:6.3, ROE:4.0 玻纤（玻璃纤维）是建筑/汽车/风电的通用材料 → 玻纤价格从2022年高点6000元/吨跌至2024年4000元/吨（-33%）→ 巨石成本控制系统良好（天然气+电耗的能源成本优化、池窑大型化降低折旧）→ 在同行亏损时仍能维持5-8%净利率 → 但当前库存水平仍偏高（约2个月产销比），行业走出低谷需等风电/光伏安装需求（如玻纤增强叶片）的明确上行拐点。格雷厄姆打分：1/5。
