行业深度分析

6.3 航空运输

航空业是沪深300中对宏观经济敏感度最高的行业之一，因其同时暴露于油价、汇率、居民消费信心和地缘政治四大外部变量。三大航和上海机场的合计权重约1%，但分析价值极高。

16. 南方航空（600029）—— PE:16.49, PB:2.64, ROE:4.07%

航油成本是航空公司最大的单一运营成本，通常占营业成本的30-35%。传导路径为：国际原油价格（布伦特）波动 → 国内航油出厂价（每季度调整一次，机制炼油成本+国际油价联动）→ 南航每年消耗航油约900-1000万吨 → 航油价格每上涨10%，南航的年度航油成本增加约30-40亿元 → 相当于2025年净利润的约50-60%。2024-2026年布伦特油价在70-90美元/桶区间波动，虽然低于2022年的120美元高点，但远高于2020年的30-40美元，意味着南航的航油成本较疫情前仍高出约40-50%。南航通过燃油套期保值来部分对冲油价风险，但套保规模有限（约20-30%的用油量）。

中国航空公司的飞机采购和融资租赁多以美元计价，形成巨额美元负债（南航美元债务约50-60亿美元等值）。传导路径为：人民币兑美元汇率波动 → 南航的美元债务按期末汇率折算产生汇兑损益 → 人民币每贬值1%（如从7.2贬至7.27），南航的汇兑损失约4-6亿元人民币 → 直接影响净利润（2025年人民币对美元贬值约2%对南航的汇兑影响约10亿元）。这一链条使航空股成为"人民币汇率敏感股"——人民币贬值，三大航利润必然受损；人民币升值，三大航利润显著受益。

航空需求是居民消费信心的晴雨表。传导路径为：GDP增速+居民可支配收入增速 → 商务出行需求（差旅预算松紧）→ 旅游出行需求（休闲消费意愿）→ 旅客周转量（RPK）变动 → 客座率（Load Factor）升降 → 单位座公里收入（RASK）变动 → 最终体现为国航票价水平。2024-2026年中国的国际航线恢复率从疫情后的约50%恢复至约80-90%，但国内航线运力过剩导致票价承压。中国对多国实施免签政策（如2024年底对日本免签）→ 国际旅客入境增长→ 推动国际航线收益提升。

格雷厄姆视角：南航PE 16.49倍、PB 2.64倍、ROE 4.07%。航空业在格雷厄姆框架下通常不属于首选，原因包括：（1）盈利波动性极大（经济衰退期往往亏损），（2）高固定成本（飞机折旧/租赁）导致的经营杠杆使利润对收入的敏感度极高，（3）油价和汇率两大不可控变量使盈利预测困难。格雷厄姆可能只建议在对油价和汇率都有明确判断的情况下才考虑航空股。

19. 上海机场（600009）—— PE约40-50x, PB约2.5-3.0x, ROE约5-6%

上海机场（浦东机场）的商业模式本质上是"流量变现"——将机场的旅客流量通过免税店、商业租赁、广告等方式变现。其中免税店租金是最大的利润来源（占营业利润的约60-70%）。传导路径为：国际旅客吞吐量恢复 → 免税店（日上免税行）的客流量和客单价提升 → 按照保底租金+销售提成（约40-45%）的模式计算 → 免税租金收入增长。2025年浦东机场国际旅客量恢复至2019年的约85%，免税租金对应的利润水平也修复到了约2019年的70-75%。如果国际旅客量恢复至2019年水平的100%+（考虑中国出境游的长期增长趋势），免税租金仍有约30%的上行空间。

2021年上海机场与日上免税行重新签署了租金合同，从"保底+高提成"改为"更灵活的保底+根据客流量分档提成"。传导路径为：2020-2022年疫情期间国际旅客归零 → 日上免税行无力支付高保底租金 → 双方重新谈判 → 新合同降低了保底水平、提高了与客流量挂钩的浮动比例 → 意味着在国际旅客恢复阶段，上海机场的免税收入弹性低于原合同（但下限得到保护）。如果2027-2028年国际旅客超预期增长，上海机场可能会重新谈判更有利的合同条款。

中国政府对海南离岛免税政策的持续放松（额度从3万提至10万、品类扩大、提货便利化）正在分流机场免税渠道的销售额。传导路径为：消费者直接从海南购买免税品（无需出国）→ 机场免税渠道的客单量（平均每人消费金额）从2019年的约2500元降至2025年的约1800元 → 即使客流恢复，免税总收入可能无法回到2019年水平。这是上海机场面临的长期结构性挑战——需要在"流量恢复"和"客单价下降"之间博弈。

主题四：人民币汇率——出海企业的隐形利润变量

原理：这些公司的美元收入不变，但折算为人民币后增加。同时，其成本（原材料、人力、水电）以人民币计价，不受汇率直接影响。收入端升值的"翻译效应"直接转化为利润增厚。

2024-2026年，中国央行为了刺激经济增长（应对房地产市场下行），实施了较宽松的货币政策，人民币从6.8贬值至7.3区间。这一趋势如果延续，对上述企业形成持续利好。但如果人民币意外升值（如中美贸易协议推动），则是负面因素。

12. 南方航空(600029) -- 强周期股，适合波段而非持有

当前估值：股价5.39元 | PE(TTM) 31.66倍 | PB 2.64倍 | 52周高6.01元/低4.91元 | 总市值977亿元

疫情三阶段：2020年疫情冲击跌至约4元→2023年"报复性出行"反弹至约8元→2024-2026年又跌回5元。典型的"V型反弹+二次回落"。

驱动因素三角模型：①油价(负相关)：航空燃油占总成本约30-35%，油价每涨10%，利润影响约20-30亿；②汇率(负相关)：南航持有大量美元债(约150亿美元)，人民币每贬值1%，汇兑损失约10-15亿；③客座率/票价(正相关)：供需决定利润弹性。2025年首次扭亏(净利润8.57亿)，2026Q1盈利14.8亿(Q1通常为淡季)，基本面在改善。

历史规律：航空股极少能"长期持有"赚钱——2007年至今持有南航几乎零回报。更适合在周期底部(油价低位+汇率稳定+需求复苏)买入做波段，或在"繁荣周期"(客座率>85%+油价<60美元)持有。

当前周期位置：处于盈利恢复初期(刚扭亏)。PE 31倍看起来不便宜，但要理解——航空股必须在利润全面恢复(全年净利50-80亿)后才能消化估值。对应PE约12-20倍才是合理区间。
